# 🧠 Quiz App (JavaScript)

This is a simple and interactive Quiz Application built using Vanilla JavaScript.  
The app displays multiple-choice questions, includes a countdown timer, and calculates the final score at the end.

The purpose of this project is to practice JavaScript fundamentals and build a small interactive web application.

## 🚀 Features

- Multiple choice questions
- Timer for each question (15 seconds)
- Automatic score calculation
- Result screen with restart option
- Clean and responsive design

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript (DOM Manipulation & Timer)

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

No installation required.
